## 🔧 必须优化项

### 1. 数据库测试数据清理机制
- **问题**: 用户服务测试中存在唯一约束冲突
- **原因**: 测试数据未在测试后清理
- **建议**: 添加测试前后的数据库清理钩子

### 2. 压力测试环境配置
- **问题**: 压力测试无法连接到开发服务器
- **原因**: 端口配置或网络设置问题
- **建议**: 配置专用测试环境，确保测试脚本与服务器正常通信

### 3. selectLighting函数优化
- **问题**: 部分场景描述未正确匹配光照类型
- **原因**: 关键词匹配规则不够完善
- **建议**: 扩展光照类型匹配规则，增加更多场景关键词
