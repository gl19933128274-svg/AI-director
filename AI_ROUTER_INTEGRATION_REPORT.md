## ⚙️ 3. 生产配置更新

### 3.1 环境变量更新

| 变量 | 值 | 说明 |
|------|------|------|
| USE_REAL_AI | true | 启用真实 AI |
| AI_PRIMARY_MODEL | auto | 自动选择主模型 |
| AI_FALLBACK_ENABLED | true | 启用 fallback |
| AI_MAX_RETRIES | 3 | 最大重试次数 |
| AI_TIMEOUT_MS | 60000 | 请求超时时间 |

### 3.2 API Routing 配置

```yaml
routes:
  storyboard:
    primary: hunyuan
    fallback: local
  video:
    primary: kling
    fallback: hunyuan
  text:
    primary: hunyuan
    fallback: -
```
