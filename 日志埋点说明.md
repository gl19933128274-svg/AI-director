
## 四、日志查看方法

### 浏览器控制台
1. 打开浏览器开发者工具 (F12)
2. 切换到 **Console** 标签
3. 筛选日志：在控制台输入 `[LOG]` 或使用过滤器

### 日志级别说明
- `[LOG]` - 普通日志（流程追踪）
- `[WARN]` - 警告日志（潜在问题）
- `[ERROR]` - 错误日志（异常情况）

| 分镜数据丢失 | `加载项目数据`、`保存分镜` |
| 视频生成失败 | `开始视频生成`、`生成中`、`生成完成` |
| 下载失败 | `下载完成` |

## 六、日志扩展建议

1. **用户追踪**：添加用户ID或会话ID用于用户行为分析
2. **性能监控**：添加时间戳计算各步骤耗时
3. **错误上报**：对接Sentry等错误监控服务
4. **结构化日志**：输出JSON格式便于日志系统解析
5. **环境区分**：开发环境输出详细日志，生产环境简化
